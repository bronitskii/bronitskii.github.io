Home Network:

          

Infrastructure & Resilience Engineering:

I architect and maintain a distributed cloud-native infrastructure designed for high availability and extreme environmental resilience.

              Operating 3 homelab servers in total, my stack leverages Docker, Caddy, and Cloudflare to host critical services like Vaultwarden and Komodo. To mitigate power grid instability in Ukraine, I engineered a custom state-synchronisation workflow: my server in Ukraine periodically polls API deployed on my other server to synchronise schedules for power outages. This data triggers automated ACPI S5 (Power Off) and rtcwake sequences, enabling unmanned, graceful shutdowns and hardware-level scheduled startups, with real-time status orchestration via ntfy push notifications.



Practical Automation:

I'm a strong advocate for efficiency through automation, I develop custom automated solutions to streamline my daily routine. To name a few projects, I have a barcode-based inventory system integrated via n8n, a context-aware iOS workflow that dynamically adjusts alarms based on calendar events, location, and live commute times; and a system that auto-schedules barber appointments based on mutual availability syncing with my calendars.



Security Architecture & Experimentation:

Security is the foundational layer of my engineering philosophy. I implement a Zero Trust multi-layered defense strategy, utilizing YubiKeys for hardware-backed MFA and Post-Quantum Hybrid SSH authentication (mlkem768x25519-sha256). My network is hardened via OpenWRT segmentation, configured with VLESS for obfuscated tunneling, default-drop firewall rules and different subnets for isolation ensuring Least Privilege access. Additionally, I operate a public DNS-over-HTTPS service that applies key-based hashing for query protection, with access additionally controlled via password‑enforced whitelist authentication
